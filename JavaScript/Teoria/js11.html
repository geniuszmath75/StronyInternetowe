Słowo kluczowe const:

1. Deklarowanie zmiennych przy pomocy const:

a) Przykład 1:

Składnia: 

var x = 10;
"Tutaj x jest równe 10"

{
    const x = 5;
    "Tutaj x jest równe 5"
}
"Tutaj x jest równe 10"
document.getElementById("c1").innerHTML = x;

Wynik:

b) Przykład 2:

Składnia: 

try {
    const PI = 3.141592653589793;
    PI = 3.14;
}
catch (err) {
    document.getElementById("c2").innerHTML = err;
}

Wynik:

Składnia: 

"Utworzenie obiektu:"

Funkcja strzałkowa:

1. Różnica pomiędzy zwykłą funkcją a funkcją strzałkową:

a) Zwykła funkcja:

Składnia: 

var hello;

hello = function() {
    return "Cześć wszystkim!";
}

document.getElementById("zf").innerHTML = hello();

Wynik:

b) Funkcja strzałkowa:

Składnia: 

var hello;

hello = () => {
    return "Cześć wszystkim!"
}

document.getElementById("fs").innerHTML = hello();

Wynik:

2. Przykłady użycia:

a) Przykład 1:

Składnia: 

var hello;

hello = (war) => "Cześć " + war;

document.getElementById("fs1").innerHTML = hello(" i na razie");

Wynik:

a) Przykład 2 (zwykła funkcja):

Składnia: 

var hello;

window.addEventListener("load", hello);

document.getElementById("btn").addEventListener("click", hello);
