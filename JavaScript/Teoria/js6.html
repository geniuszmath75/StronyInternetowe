Daty:

1. Wyświetlanie aktualnej daty:

Składnia: 

var d = new Date();
document.getElementById("data").innerHTML = d;

Wynik:

2. Sposoby zapisywania dat:

Składnia:

a) new Date() - aktualna data i czas
b) new Date(rok, miesiąc, dzień, godziny, minuty, sekundy, milisekundy) - określona data i czas
c) new Date(milisekundy) - czas w milisekundach
d) new Date(dateString) - data zapisana stringiem 

a) var a = new Date();
document.getElementById("data").innerHTML = a;

b) var b = new Date(2020, 7, 7, 15, 30, 0, 0)
document.getElementById("data").innerHTML = b;

c) var c = new Date(2020)
document.getElementById("c").innerHTML = c;

d) var d = new Date("October 13, 2014 11:13:00");
document.getElementById("d").innerHTML = d;

Wynik:

3. Sposoby wyświetlania dat:

a) toUTCString():

Składnia: 

var d = new Date();
document.getElementById("date").innerHTML = d.toUTCString();

Wynik:

b) toDateString():

Składnia: 

var d = new Date();
document.getElementById("date1").innerHTML = d.toDateString();

Wynik:

c) toISOString():

Składnia: 

var d = new Date();
document.getElementById("date2").innerHTML = d.toISOString();

Wynik:
