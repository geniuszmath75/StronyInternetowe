Pętla for:

1. Składnia ogólna: 

for (instrukcja 1; instrukcja 2; instrukcja 3) {
   // blok kodu, który ma być wykonany>
}

a) Instrukcja 1 jest wykonywana (jednorazowo) przed wykonaniem bloku kodu.

b) Instrukcja 2 definiuje warunek wykonania bloku kodu.

c) Instrukcja 3 jest wykonywana (za każdym razem) po wykonaniu bloku kodu.

2. Przykład 1:

Składnia: 

var text = "";
var i;

for (var i = 0; i < 11; i++) {
   	text += "Liczba to " + i + "
";
}

document.getElementById("p1").innerHTML = text;

var samochody = ["BMW", "Volvo", "Nissan", "Ford"];

Pętla while:

1. Składnia ogólna: 

while (warunek) {
   // blok kodu, który ma być wykonany>
}

2. Przykład 1:

Składnia: 

var text = "";
var i = 0;

while (i < 11) {
   	text += "Liczba to " + i + "
";
   	i++;
}
document.getElementById("w1").innerHTML = text;

Wynik:

3. Pętla do/while:

Składnia: 

var text = "";
var i = 0;

do {
   	text += "Liczba to " + i + "
";
   	i++;
}
while (i < 11);

document.getElementById("dow").innerHTML = text;

Break i Continue:

1. Break:

Składnia: 

var text = "";
var i;

for (i = 0; i < 10; i++) {
    if (i === 3) { break; }
    text += "Liczba to " + i + "
";
}
document.getElementById("br").innerHTML = text;

Wynik:

2. Continue:

Składnia: 

var text = "";
var i;

for (i = 0; i < 11; i++) {
    if (i === 3) { continue; }
    text += "Liczba to " + i + "
";
}
document.getElementById("con").innerHTML = text;
